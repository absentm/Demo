## RecyclerViewDemo

Android RecyclerView 使用 Demo, 包涵了如下功能：
1. RecyclerView 控件的基本用法 ;
2. item 的 Click, LongClick 事件处理 ;
3. 同一样式的 Item 不同布局的展示 ;
4. Item 控件的状态变化处理, 如阅读痕迹等 ...
